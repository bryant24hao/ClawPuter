- [ ] 电量显示 — 屏幕角落电池百分比，低电量角色变虚弱
- [ ] 亮度自动调节 — 夜间自动降亮度

## Phase 6：互动玩法

- [ ] 养成系统 — 饥饿值/心情值，需定期"喂食"，否则角色变蔫
- [ ] 表情反应 — AI 回复关键词（happy/sad/wow）触发对应表情
- [ ] 小游戏 — 按键接物 / 打地鼠，小龙虾钳子接东西

## Phase 7：实用功能

- [ ] 番茄钟 — 25 分钟倒计时，角色陪你专注，到时间蹦起来提醒

- [ ] 提醒事项 — 键盘输入提醒，到点角色弹窗
