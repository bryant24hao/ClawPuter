# 产品迭代路线图

## Phase 4 ✅ (已完成)

- [x] 开机动画 — 像素小龙虾逐行渐入
- [x] 模式切换动画 — 像素擦除过渡
- [x] 日夜背景 — 根据 NTP 时间切白天/傍晚/夜晚
- [x] 角色自发动作 — 伸懒腰、东张西望
- [x] 音效 — 按键咔嗒、开心音阶、AI 回复提示音
- [x] SSE 流式响应 — 逐 token 显示 AI 回复，零堆分配 chunked transfer 解析
- [x] 语音输入 — Fn 按住说话，Groq Whisper STT 转文字，本地代理中转
- [x] 桌面宠物同步 — macOS 端 Swift 桌面宠物，通过 UDP 接收 Cardputer 状态
- [x] OpenClaw 集成 — 局域网连接 OpenClaw Gateway，多模型自动切换

## Phase 4.5 ✅ (已完成)

- [x] 运行时可配置 Gateway/STT — 编译时值作为默认，Fn+R 向导可修改，重启后保持
- [x] 备用 WiFi 自动降级 — 主 WiFi（家里）连不上自动尝试备用（手机热点），编译时配置两套
- [x] 备用 Gateway Host — 连上备用 WiFi 时自动切换 Gateway IP（热点网络 Mac IP 不同）
- [x] Setup 向导可取消 — Tab 退出向导回到 Companion 模式
- [x] WiFi 连接失败不死循环 — 所有 WiFi 都失败后提供选择：重试 / 进 Setup 改配置 / 跳过进入离线模式
- [x] 离线模式 — 跳过联网后 Companion 模式正常可用（动画、时钟、音效），Chat 模式显示离线提示

## Phase 5 ✅ (已完成)

- [x] TTS 语音回复 — AI 回复通过扬声器播放
- [x] 宠物键盘移动 — `;./ ,` 方向键，按住持续走，精灵翻转朝向
- [x] 时光倒流天空 — 宠物 X 位置映射 ±12h 时间偏移，背景随位置变化（左=过去，右=未来），地面时钟仍显示真实时间
- [x] 桌面位置同步 — UDP 广播扩展 x/y/d 字段，Cardputer 移动宠物 → 桌面宠物同步移动

## Phase 6：体验打磨

- [ ] 电量显示 — 屏幕角落电池百分比，低电量角色变虚弱
- [ ] 亮度自动调节 — 夜间自动降亮度
- [ ] 聊天历史持久化 — 存 NVS 或 SD 卡，重启后保留上次对话

## Phase 7：互动玩法

- [ ] 小游戏 — 按键接物 / 打地鼠，小龙虾钳子接东西

- [ ] 提醒事项 — 键盘输入提醒，到点角色弹窗
